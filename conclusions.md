# CONCLUSIONS OF THE SIMULATIONS

## Object

Cylinder with radius 12*lamda with different discretization. 



## FRESNEL REGION 

In this region the Frahonfer condition is not satisfied but the condition r >> lamda is. 
The scattering in this region is better calculated with

###	 E^2/R = 3.43 => RMSE circa 1.5 dB

Discretization is 12




## NEAR FIELD

When the condition r >> lamda is not satisfied a low discretization is required to decrease the RMSE.

### E^2/R = 7.50 => RMSE circa 2.0 dB

Discretization is 8
